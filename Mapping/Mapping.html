Peta Geomorfologi Kota Malang

Peta Geomorfologi Kota Malang yang telah saya buat menggunakan sumber data dari peta jenis tanah skala 1:50.000 dan peta admin dukcapil 2019 serta pengolahan dilakukan dengan perangkat lunak ArcGIS 10.5 untuk jadi bagian penelitian skripsi saya.

Peta Tutupan Lahan Kota Malang

Pembuatan peta tutupan lahan Kota Malang menggunakan citra satelit Landsat 7 dan 8 OLI yang di olah melalui Google Earth Engine, kemudian layout di ArcGIS 10.5.

Peta Suhu Permukaan Tanah Kota Malang

Pembuatan peta tutupan lahan Kota Malang menggunakan citra satelit landsat 7 & 8 OLI yang didapatkan melalui Climate Engine, kemudian layout di ArcGIS 10.5.

Peta Kesesuaian Penggunaan Lahan Kota Pasuruan Pada Kecamatan Bugul kidul menurut RTRW

Pembuatan peta kesesuaian lahan kota pasuruan menurut RTRW bersumber dari lampiran RTRW Kota Pasuruan Tahun 2011-2031, kemudian dilakukan layout di ArcGIS 10.5.

Peta Kerawanan Longsor Kecamatan Jabung Malang

Pembuatan peta kerawanan longsor ini merupakan project dari mata kuliah Survei Pemetaan Geologi dan Geomorfolgoi, data peta bersumber dari Peta RBI Nongkojajar, Peta Geologi lembar bromo, Peta Kemiringan Lereng, Peta penggunaan Lahan, peta bentuklahan, dan data analisis tekstur tanah lokasi kajian. Kemudian di gabung menggunakan perangkat lunak ArcGIS 10.5.

Survei Pengukuran Volume dan Kecepatan Air Sungai

Survei lapangan ini saya membantu teman saya dalam mengerjakan penelitian skripsinya, lokasi survei berada di Kota Batu.

Survei Pengukuran Rel Kereta Api Milik PT KAI

Pengukuran titik koordinat as rel kereta api dilakukan dengan menggunakan alat GPS Geodetik/GNSS dan di dampingi oleh pihak dari PT KAI.

Pengukuran Batas Lahan Kebun Petani di Wilayah Hutan Pehutani Gunung Arjuna

Pengukuran titik koordinat batas lahan kebun menggunakan GPS smartphone; Avenza Maps dan di dampingi oleh para petani serta pihak perhutani.
